Short Guide

To use ProcessManager, just copy all files to your project's directory and include ProcessManager.h

#include "ProcessManager.h"

There are two ways to use class ProcessManager

  • Creating new process from executable

    • Specify path to process executable and command line arguments, and then call startProcess() method to create process

    ProcessManager p = ProcessManager(L"c:/windows/notepad.exe",L"a.txt");

    p.startProcess();

    To do some operations you should use loops such this

    ProcessManager p = ProcessManager(L"c:/windows/notepad.exe",L"a.txt");

    p.startProcess();

    int swch;

    while(1)

    {

    //...entering swch

     switch(swch) {

      case 1: p.pauseProcess(); break;

      case 2: p.resumeProcess(); break;

      case 3: p.restartProcess(); break;

     /* .....*/

      default: //...

     }

    }

  • Openning currently running process

    • To open current running process, you should specify id of process

    ProcessManager p = ProcessManager(1234);
