Детальний опис

Logger class defined for logging ProcessManager events. It use Windows event logging for making logs about all actions an errors caused by ProcessManager Logger uses event_cat.h constants that were generated and compiled by mc tool from event_cat.mc to identify massage type and id. Also it contains a set of string message templates that are used to send logs to Event Log
