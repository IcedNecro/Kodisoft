	/html - папка, де міститься згенерована doxygen документація. Загалом, там 	
	надана вичерпна інформація про мою реалізацію завдання. В корневі репозиторію 	
	є ярлик для зручностіщоб не шукати index.html всередині папки
	
	/KodisoftTest - папка з сирцями, для того щоб використовувати класи, достатньо
	просто скопіювати їх в папку проекта і директивою підключити хедер 
	
	#include "ProcessManager.h"

==========================================================================================
||				Опис файлів висхідного коду				||
==========================================================================================
	
	Коротко по кожному файлу кода:

	/KodisoftTest/ProcessManager.h
		Файл класу з описом методів, що треба описати згідно з завданням

	/KodisoftTest/ProcessManager.cpp
		Файл реалізація методів класа

	/KodisoftTest/Logger.h
		Класс для ведення логування в Event Log
	
	/KodisoftTest/Logger.cpp
		Реалізація класу логгера
	
	/KodisoftTest/event_cat.h
		хедер, що містить коди повідомлень, які розуміє Event Log. Компілюється з
		event_cat.mc
	
	/KodisoftTest/event_cat.mc
		вихідний файл для компілятора повідомлень

			Служебні файли, наукового інтересу не представляють
	/KodisoftTest/Process.h
	/KodisoftTest/Process.cpp
	/KodisoftTest/targetver.h
